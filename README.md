## 📦 Project Architecture Breakdown

### 1. Interactive Calculator (`/calculator`)
A functional, browser-based arithmetic calculator engineered with immediate input rendering capabilities.
* **Core Mechanisms:** Direct string tokenization evaluations, edge-case math processing (e.g., zero divisions), and unified button node matrix input listeners.
* **Stack:** HTML5, CSS3 Grid Flexbox structures, Javascript Event Listeners.

### 2. Dynamic Split-Column To-Do Tracker (`/todo-app`)
A task management workspace designed to split entries into distinct processing columns for tracking daily routines.
* **Core Mechanisms:** Real-time array data hydration, structural DOM element sorting nodes, task state toggles, and dynamic numeric column counters.
* **Stack:** HTML5, CSS3 dark surface layout variables, Javascript Lifecycle hooks.

### 3. Engineering Icon Tribute Sheet (`/tribute`)
A dark-mode single-page documentation app layout dedicated to the creator of Linux and Git, Linus Torvalds.
* **Core Mechanisms:** Accessible semantic tagging matching structural container bounds, low-poly wireframe vector art integration, and responsive chronological timeline nodes.
* **Stack:** Semantic HTML5 (`<main>`, `<section>`, `<figcaption>`), Fluid Typography CSS layout sheets.

### 4. Secure Gateway Authentication Engine (`/login-validation`)
A secure dual-purpose portal allowing seamless runtime view transformations between user sign-in interfaces and validation sign-up cards.
* **Core Mechanisms:** Interception loops mapping custom `e.preventDefault()` barriers, regex email string pattern filters, length-constraint password loops, verification comparisons, and non-blocking view changes.
* **Stack:** Advanced CSS custom variables, Tab-state toggle selectors, Javascript client-side validation logic.
